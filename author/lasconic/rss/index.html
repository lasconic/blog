I'm using Markdown and git daily, so an engine based on them would be nice. So no good solution... <p>I came accross <a href="https://ghost.org/download/">Ghost</a> which seems to gain momentum. Ghost lets me write in Markdown syntax and is based on Node.js. I'd prefer to use Github Pages instead, so static HTML pages.</p> <p><a href="https://github.com/axitkhurana/buster">Buster</a> to the rescue! Buster (Ghost, Buster, <a href="https://www.youtube.com/watch?v=I9bNyDjczVQ">got it</a>?) is a Python script to extract static content from a running Ghost instance. OK! Download the zip from <a href="https://ghost.org/download/">Ghost download page</a>.
